Q. What is DOM and purpose of DOM?

DOM is stand for 'Document Object Model' it's a language-independent interface that treat HTML file as a tree structure. Where each element is a Node. The purpose of DOM is manipulate, style, create dynamic data in document.

Q. How to select element(s) with DOM?

There are couple of way to select element with DOM but the most popular or most uses ways are getElementById(), getElementsByClassName(), getElementByTagName(), querySelector() finally querySelectorAll(). Also there are others way but these are the most popular ways.

Q. What is Event Bubbling?

Event Bubbling is a concept in the DOM. Where event an element receives bubbled to it's parent and ancestors, upward in the tree till gets to the root.

Q. What is call-back function?

Call-back function is, passed a function into another function as an argument. And it will execute after another function finished execution.
